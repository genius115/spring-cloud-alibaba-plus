##二、容器

```
2.1创建容器
docker run -it -d -p 8881:8881 --name examplenacos1 example/nacos:1.0 --spring.cloud.nacos.config.server-addr=39.107.77.144:18848 --spring.cloud.nacos.discovery.server-addr=39.107.77.144:18848

2.2指定网络为宿主机，不用映射端口
docker run -it -d  --network=host --name examplenacos1 example/nacos:1.0
注：
1、容器主机集群化技术
consul注册中心

2、容器网络模式  overlay 
host模式：使用 --net=host 指定。
none模式：使用 --net=none 指定。
bridge模式：使用 --net=bridge 指定，默认设置。
container模式：使用 --net=container:NAME_or_ID 指定
```
